# Addiction & Hunger Lexicon

**Purpose**: Provide rich, modular vocabulary for hunger, addiction, eating behavior, withdrawal, and desperate craving states.

All entries are designed to work with the existing Modular Text System.

---

## 1. Hunger Descriptors (`word.hunger`)

```js
export const HUNGER_WORDS = {
  normal: [
    "a normal appetite",
    "her usual hunger",
  ],
  increased: [
    "noticeably hungrier than usual",
    "clearly thinking about food more often",
    "her stomach growling more than normal",
  ],
  high: [
    "very hungry",
    "her stomach growling loudly",
    "visibly distracted by hunger",
  ],
  craving: [
    "craving food",
    "restless with hunger",
    "clearly fighting the urge to eat",
    "stomach twisting with hunger",
  ],
  starving: [
    "starving",
    "desperate for food",
    "visibly struggling with hunger",
    "looking almost frantic with hunger",
  ],
};
```

## 9. Suggested Module Structure

Here’s how you can register these in the engine:

```js
registerModule("word.hunger", byBucket(HUNGER_WORDS));
registerModule("word.addictedHunger", byBucket(ADDICTED_HUNGER_WORDS));
registerModule("word.eating", byBucket(EATING_STYLE));
registerModule("word.addictedEating", byBucket(ADDICTED_EATING));
registerModule("word.begging", byBucket(BEGGING_WORDS));
registerModule("word.withdrawal", byBucket(WITHDRAWAL_BEHAVIOR));

registerModule("hunger.desc", [ /* tier logic */ ]);
registerModule("hunger.addictedDesc", [ /* addiction-flavored hunger */ ]);
registerModule("eating.style", [ /* normal vs addicted eating */ ]);
registerModule("talk.hungryCoda", [ /* dialogue additions */ ]);
registerModule("scene.hungerInterrupt", [ /* interrupting scenes */ ]);
```

---

## 10. Quick Usage Examples

**Diary Entry:**
```js
"I can’t stop thinking about food. {hunger.desc|cap}. {hunger.addictedDesc|cap}"
```

**Eating Scene:**
```js
"She {eating.style} while you feed her. {hunger.addictedDesc|cap}"
```

**Dialogue Coda:**
```js
"{talk.hungryCoda|prefix: }"
```

**Interrupting Scene:**
```js
"{scene.hungerInterrupt|pick} She’s standing there looking {word.begging}."
```
